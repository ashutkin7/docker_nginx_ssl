# Nginx в Docker с поддержкой SSL

Проект для быстрого развертывания веб-сервера Nginx в Docker-контейнере с поддержкой HTTP, HTTPS и возможностью горячей перезагрузки SSL-сертификатов.

## Структура проекта

- **nginx/** — конфигурационные файлы и статика веб-сервера.
  - Dockerfile — описание сборки Docker-образа.
  - index.html — статическая HTML-страница.
  - nginx-conf/nginx.conf — конфигурация хоста (порты 80/443, SSL, пути к сертификатам).
- **certs/** — директория для хранения SSL-сертификатов (монтируется как Volume).
- **deploy.sh** — скрипт автоматической генерации начальных сертификатов, сборки и запуска контейнера.
- **generate_and_reload_certs.sh** — скрипт для перегенерации сертификатов и их безопасного обновления внутри запущенного Nginx без простоя.

## Инструкция по использованию

### 1. Запуск проекта
Запустите скрипт деплоя. Он сгенерирует самоподписанные сертификаты (если они отсутствуют), соберет образ и запустит контейнер `nginx-cont`:
```bash
./deploy.sh
```
После запуска:
- **HTTP** доступен на: http://127.0.0.1:54321
- **HTTPS** доступен на: https://127.0.0.1:54322

### 2. Обновление SSL-сертификатов
Для перегенерации сертификатов и перезагрузки конфигурации Nginx без остановки веб-сервера выполните:
```bash
./generate_and_reload_certs.sh
```
